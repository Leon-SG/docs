## Replacing or adding images

- To replace: overwrite the file with the same name and extension.
- To add: use the same naming (lowercase, numbers, hyphens) and add a corresponding `![Caption](/images/user-manual/...)` and caption in `guides/user-manual.mdx` if you want it in the doc.
- After changes, rebuild or refresh the docs site so images appear.
