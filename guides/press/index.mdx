## What you can do

- Create new articles (press releases, blog posts, summaries).
- Manage drafts vs published content in a single library.
- Apply consistent tone, audience targeting, and SEO keywords.
- Refine sections without regenerating the whole document.

## Where to find it

- Open `Press` from the main navigation.

## Library basics

- **Search and filter**: find documents by keyword and status (**Draft** / **Published**).
- **Metadata**: each entry shows title, audience, date, and status.
- **New Article**: starts a new draft.
